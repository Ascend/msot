#### 参数重放流程图

``` plantuml
@startuml
start
:计算用户参数总长度;
:调用 aclrtKernelArgsGetMemSizeImpl 获取实际需要的参数缓冲区大小;
:根据大小分配参数缓冲区;
:调用 aclrtKernelArgsGetHandleMemSizeImpl 获取需要的句柄缓冲区大小;
:根据大小分配句柄缓冲区;
:调用 aclrtKernelArgsInitByUserMemImpl 初始化 argsHanlde 句柄;
group "参数重放" {
  repeat
    switch (参数类型)
    case (normal 参数)
      :调用 aclrtKernelArgsAppendImpl 追加参数;
    case (placeholder 参数)
      :调用 aclrtKernelArgsAppendPlaceHolderImpl 追加参数;
    endswitch
  repeat while (遍历记录的参数操作)
  repeat
    if (placeholder 参数) then (yes)
      :调用 aclrtKernelArgsGetPlaceHolderBufferImpl 获取 buffer;
      :将数据拷贝到 buffer;
    else (no)
    endif
  repeat while (遍历记录的参数操作)
}
:调用 aclrtKernelArgsFinalizeImpl 结束参数生成;
end
@enduml
```
